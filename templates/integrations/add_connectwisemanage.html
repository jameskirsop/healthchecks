Connectwise Manage

ConnectWise Manage provides IT Service Management, including ticketing. You can integrate it with your {{ site_name }} account in a few simple steps.

Setup Guide

Log into your ConnectWise Manage account as an administrator and go to the System › Members page.

Select the API Members tab at the top of the page.

Add a New Member permission, make sure it is checked.

The integration does not need the Read Access and Delete Access checkboxes – make sure they are unchecked.

Give the integration a descriptive name, take note of its API Key, then save the integration.

Paste the API Key down below, and make sure the correct service region is selected. Save the integration, and you are done!
